参数化测试

允许使用不同的值运行相同的测试

通过@DataProvider传递参数

@DataProvider注解的方法返回对象数组，也允许使用迭代器，此时测试方法会一个接一个地调用迭代器返回的值。e.g. Iterator<Object[]>

如果 DataProvider 和 test 方法不在同一个类中，需要在 test 方法注解中显式指明：

@Test(dataProvider="", dataProviderClass="xx.class")
