Jquery

ajax 返回错误

ajax 执行成功，但无返回值。用 alert 测试时发现进入 error 方法体内。

原因：相应的方法忘了写 @ResponseBody 注解

Ajax 进入 error 内其他几种可能原因：

  • 返回数据类型不是 json 类型
  • 请求域与当前域不一致
